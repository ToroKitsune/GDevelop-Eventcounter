# GDevelop Event Counter

This tool counts all events inside a GDevelop project JSON file.

## How to Use (Windows)

1. Install Python: https://www.python.org/downloads/
2. Download the ZIP of this project from GitHub.
3. Right‑click the ZIP → "Extract All..." → choose a folder.
4. Open the extracted folder.
5. Make sure the `Resources` folder is next to `eventcounter.py`.
6. Open a terminal in that folder:
   - Click the folder’s address bar
   - Type: cmd
   - Press Enter
7. Run:

   python eventcounter.py

8. Select your GDevelop project JSON file (usually `game.json`).
9. Click "Run Event Count" to view totals.
